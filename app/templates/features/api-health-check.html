Genel Bakış

API Health Check, REST API endpoint'lerinizin sağlık durumunu otomatik olarak kontrol eder. JSON formatında endpoint listesi alarak her birinin erişilebilirliğini ve yanıt sürelerini test eder.

Desteklenen Formatlar

Basit Liste

Endpoint'lerin basit listesi

["url1", "url2", "url3"]
Obje Listesi

Endpoint'lerin detaylı tanımı

[{"name": "API1", "url": "url1"}]
Postman Collection

Postman collection export'u

{"info": {}, "item": []}

Özellikler

Health Check
  • HTTP Status Kontrolü: 200, 404, 500 gibi durum kodlarını kontrol eder
  • Yanıt Süresi: Her endpoint'in yanıt süresini ölçer
  • Erişilebilirlik: Endpoint'lerin erişilebilir olup olmadığını test eder
  • Paralel Test: Birden fazla endpoint'i aynı anda test eder
Raporlama
  • JSON Rapor: Detaylı JSON formatında rapor
  • PDF Rapor: Profesyonel PDF raporu
  • Başarı Oranı: Genel başarı oranı hesaplama
  • Hata Detayları: Başarısız endpoint'lerin detaylı analizi

Avantajlar

  • Hızlı Kontrol: API'lerinizi saniyeler içinde kontrol edin
  • Görsel Rapor: Sağlık durumunu görsel olarak takip edin
  • Hızlı Tespit: Sorunlu endpoint'leri hızlıca tespit edin
  • Takım Paylaşımı: Raporları takımla paylaşın
  • Geçmiş Takibi: API sağlık geçmişini saklayın
  • Otomatik Tekrar: Düzenli kontroller için otomatik tekrar
